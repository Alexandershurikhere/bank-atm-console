# Console ATM Application

A simple console-based ATM simulator written in Java.  
This project demonstrates object-oriented programming, collections (HashMap, ArrayList), exception handling, and basic file I/O (optional).  

## Features
- Account creation (in code) with card number and PIN
- Authorization by card number and PIN
- Check balance
- Withdraw money (with validation)
- Deposit money
- Transfer between accounts
- Transaction history with timestamps
- User-friendly console menu

## Technologies used
- Java 8+
- Java Collections Framework (HashMap, ArrayList)
- java.time for date/time formatting
- Git & GitHub

## How to run
1. Clone the repository:  
  https://github.com/Alexandershurikhere/bank-atm-console/tree/main/Practice
2. Open the project in any Java IDE (IntelliJ IDEA, Eclipse, etc.)
3. Run the `Main` class (it contains sample accounts)
