# Console ATM Application

A feature-rich ATM simulator written in Java. This console application allows users to create accounts, authenticate with a card number and PIN, and perform typical banking operations such as checking balance, withdrawing/depositing money, and transferring funds between accounts. All transactions are recorded with timestamps.

## Features
- Account creation with unique card number and PIN
- Secure authorization
- Check current balance
- Withdraw money (with validation for sufficient funds)
- Deposit money
- Transfer money between accounts
- Transaction history with timestamps
- User-friendly console menu

## Technologies used
- Java 8+
- Java Collections Framework (`HashMap`, `ArrayList`)
- `java.time` for date and time formatting
- Git & GitHub for version control
